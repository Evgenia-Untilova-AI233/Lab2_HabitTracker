#  Android-проєкт: Трекер звичок (Habit Tracker)

Цей репозиторій містить навчальний Android-застосунок, розроблений з використанням **Jetpack Compose**, **ViewModel** та **StateFlow**.

Проєкт реалізує односторінковий додаток для щоденного відстеження звичок із реактивним оновленням інтерфейсу.


##  Функціональність

- Фіксований список щоденних звичок (6 елементів)
- Відмітка виконання кожної звички (tap або checkbox)
- Реактивний лічильник виконаних звичок
- Прогрес-бар виконання дня
- Кнопка «Скинути день» (обнулення всіх звичок)
- Динамічний інтерфейс на Jetpack Compose


##  Технології

- Kotlin
- Jetpack Compose
- ViewModel (Android Architecture Components)
- StateFlow
- Material 3 UI


##  Як запустити проєкт

### Через Android Studio:

1. Відкрити Android Studio
2. Обрати **Get from VCS**
3. Вставити URL репозиторію
4. Натиснути **Clone**
5. Дочекатися синхронізації Gradle
6. Натиснути **Run**


##  Логіка роботи

- Список звичок зберігається у `ViewModel`
- Стан змінюється через `MutableStateFlow`
- UI автоматично реагує на зміни стану
- Лічильник формується як похідне значення
